## 전체 화면 구조

### 화면 플로우

```
[메인 랜딩 페이지]
    ├─→ [로그인]
    │    └─→ [메인으로 복귀]
    ├─→ [회원가입]
    │    └─→ [로그인]
    ├─→ [카테고리 상세 (×5)]
    │    ├─→ 정책 상세 (미구현)
    │    ├─→ 영상 재생 (미구현)
    │    └─→ 퀴즈 풀기 (미구현)
    └─→ [청년정책카페 (준비중)]
         └─→ [메인으로 복귀]
```

### 전체 페이지 목록

| 번호 | 페이지명 | 경로 | 컴포넌트 | 인증 필요 |
|------|---------|------|----------|-----------|
| 1 | 메인 랜딩 | `/` | FigmaLandingPage.vue | ❌ |
| 2 | 카테고리 상세 | `/category/:category` | FigmaCategoryPage.vue | ❌ |
| 3 | 로그인 | `/login` | FigmaLoginPage.vue | ❌ |
| 4 | 회원가입 | `/signup` | FigmaSignupPage.vue | ❌ |
| 5 | 준비중 | `/coming-soon` | FigmaComingSoonPage.vue | ❌ |

## 공통 컴포넌트

### 1. FigmaHeader (헤더)

**위치**: 모든 페이지 상단
**높이**: 80px (데스크톱) / 64px (모바일)
**고정**: Sticky (스크롤 시 상단 고정)

#### 레이아웃

```
┌──────────────────────────────────────────────────────────┐
│ [로고: 청년있슈]  [일자리] [주거] [교육] [금융] [참여]  [로그인] [회원가입] │
└──────────────────────────────────────────────────────────┘
```

#### 구성 요소

| 요소 | 스타일 | 동작 |
|------|--------|------|
| 로고 | 32px / Bold / Navy (#1e3a8a) | 클릭 시 메인 페이지 이동 |
| 카테고리 버튼 (×5) | 16px / 일반 (#374151) / 활성 시 주황색 (#F97316) Bold | 클릭 시 해당 카테고리 페이지 이동 |
| 로그인 버튼 | Outlined / 주황색 테두리 | 클릭 시 로그인 페이지 이동 |
| 회원가입 버튼 | Filled / 주황색 배경 | 클릭 시 회원가입 페이지 이동 |

#### 모바일 버전

```
┌──────────────────────────┐
│ [☰] 청년있슈      [로그인] │
└──────────────────────────┘
```

- 768px 이하에서 햄버거 메뉴로 전환
- 사이드 드로어로 카테고리 표시

```
┌──────────────────────────────────────────────────────────┐
│ [청년있슈 소개] [이용약관] [개인정보처리방침]                        │
│                                                          │
│ 언어 선택: [한국어 ▼]                     [고객센터 문의]    │
│                                                          │

└──────────────────────────────────────────────────────────┘

#### 구성 요소

- **링크**: 소개, 이용약관, 개인정보처리방침 (클릭 시 alert)
- **언어 선택**: 드롭다운 (한국어, English)
- **고객센터 버튼**: 클릭 시 "준비중입니다" 알림
- **저작권**: 고정 텍스트

## 페이지별 상세 설계

### 1. 메인 랜딩 페이지 (`/`)

**컴포넌트**: `FigmaLandingPage.vue`
**목적**: 사용자에게 5가지 정책 카테고리 소개 및 빠른 접근 제공

#### 전체 구조

```
┌────────────────────────────────────────────────────────────┐
│ [Header]                                                   │
├────────────────────────────────────────────────────────────┤
│ [Hero Section - 자동 슬라이드]                              │
├────────────────────────────────────────────────────────────┤
│ ┌───────────────────────────┬───────────────────┐          │
│ │ [Main Content]            │ [Sidebar]         │          │
│ │                           │ (데스크톱 전용)    │          │
│ │ 청년을 위한 5가지 정책     │                   │          │
│ │                           │                   │          │
│ │ [카드1] [카드2] [카드3]   │                   │          │
│ │ [카드4] [카드5]           │                   │          │
│ └───────────────────────────┴───────────────────┘          │
├────────────────────────────────────────────────────────────┤
│ [Footer]                                                   │
└────────────────────────────────────────────────────────────┘
```

#### Hero Section (FigmaHeroSection)

- **높이**: 400px
- **기능**: 자동 슬라이드 캐러셀
- **슬라이드 간격**: 4초
- **슬라이드 수**: 3개 (news-1.png, news-2.png, news-3.png)
- **컨트롤**: 좌우 화살표 + 하단 점 네비게이션
- **이미지**: object-fit: cover

#### Service Cards (FigmaServiceCards)

##### 섹션 헤더

- **제목**: "청년을 위한 5가지 정책" (32px / Bold)
- **부제**: "청년 여러분의 꿈을 응원하는 맞춤형 정책을 확인해보세요" (16px / Gray)
- **여백**: 하단 48px

##### 카드 그리드

- **레이아웃**: Grid
  - 모바일 (< 640px): 1열
  - 태블릿 (640px ~ 1024px): 2열
  - 데스크톱 (≥ 1024px): 3열
- **간격**: 24px
- **카드 스타일**:
  - 배경: 그라디언트 (135도)
  - 모서리: 16px 라운드
  - 그림자: `0 10px 15px -3px rgba(0, 0, 0, 0.1)`
  - 패딩: 24px
  - 텍스트: 흰색
  - Hover: 그림자 확대

##### 카드별 상세

| 카드 | 제목 | 아이콘 | 그라디언트 | 설명 | 뱃지 |
|------|------|--------|-----------|------|------|
| 1 | 일자리 | work | #3B82F6 → #2563EB (파랑) | 청년 취업 지원과 창업 기회를 제공합니다 | - |
| 2 | 주거 | home | #F97316 → #EA580C (주황) | 안정적인 주거 환경을 위한 지원 정책 | - |
| 3 | 교육 | school | #4ADE80 → #22C55E (초록) | 학업과 역량 개발을 위한 다양한 지원 | - |
| 4 | 금융･복지･문화 | account_balance | #F472B6 → #EC4899 (분홍) | 경제적 지원과 문화생활 혜택 | - |
| 5 | 참여 | groups | #2DD4BF → #14B8A6 (청록) | 청년의 목소리를 정책에 반영합니다 | 인기 |

**카드 내부 구조**:

```
┌──────────────────────────────┐
│ [제목]              [아이콘]  │
│                              │
│ [설명 텍스트]                 │
│                              │
│                    [인기뱃지] │
└──────────────────────────────┘
```

- **제목**: 24px / Bold
- **아이콘**: 48px × 48px / 흰색 반투명 배경
- **설명**: 14px / 90% 불투명도
- **인기 뱃지**: 빨간색 배경 / 12px / 우상단

### 2. 카테고리 상세 페이지 (`/category/:category`)

**컴포넌트**: `FigmaCategoryPage.vue`
**목적**: 선택한 카테고리의 정책 정보 및 학습 콘텐츠 제공

#### 전체 구조

```
┌────────────────────────────────────────────────────────────┐
│ [Header]                                                   │
├────────────────────────────────────────────────────────────┤
│ [← 뒤로가기]          [카테고리 제목]                       │
├────────────────────────────────────────────────────────────┤
│ 📊 학습 진도율: 45%                                         │
│ ████████████░░░░░░░░░░░░░░░░░░ 45%                        │
├────────────────────────────────────────────────────────────┤
│ 📋 정책 목록 (3개)                                          │
│ ┌────────────┬────────────┬────────────┐                   │
│ │ [정책1]    │ [정책2]    │ [정책3]    │                   │
│ └────────────┴────────────┴────────────┘                   │
├────────────────────────────────────────────────────────────┤
│ 🎬 학습 영상 (3개)                                          │
│ ┌────────────┬────────────┬────────────┐                   │
│ │ [영상1]    │ [영상2]    │ [영상3]    │                   │
│ └────────────┴────────────┴────────────┘                   │
├────────────────────────────────────────────────────────────┤
│ ❓ OX 퀴즈 (5개)                                           │
│ ┌────────────┬────────────┬────────────┐                   │
│ │ [퀴즈1]    │ [퀴즈2]    │ [퀴즈3]    │                   │
│ │ [퀴즈4]    │ [퀴즈5]    │            │                   │
│ └────────────┴────────────┴────────────┘                   │
├────────────────────────────────────────────────────────────┤
│ [Footer]                                                   │
└────────────────────────────────────────────────────────────┘
```

#### 페이지 헤더

- **뒤로가기 버튼**: 좌측 상단 / Flat / 회색
  - 아이콘: `arrow_back`
  - 동작: 메인 페이지로 복귀
- **카테고리 제목**: 32px / Bold / 카테고리 색상
  - 예: "일자리" → 파란색 (#3B82F6)

#### 학습 진도율 섹션

```
┌────────────────────────────────────────┐
│ 📊 나의 학습 진도                       │
│                                        │
│ ████████████░░░░░░░░░░░░░░░░ 45%      │
│ 3개 중 1개 완료                         │
└────────────────────────────────────────┘
```

- **진도율 바**: Quasar Q-Linear-Progress
- **색상**: 카테고리별 그라디언트
- **높이**: 8px
- **모서리**: 4px 라운드
- **텍스트**: 완료 개수 / 전체 개수

#### 정책 목록 섹션

**섹션 제목**: "📋 일자리 관련 정책" (24px / Bold)

**카드 그리드**: 3열 (태블릿 2열, 모바일 1열)

**정책 카드 구조**:

```
┌──────────────────────────┐
│ [썸네일 이미지]           │
│                          │
├──────────────────────────┤
│ [정책 제목]              │
│                          │
│ [짧은 설명...]           │
│                          │
│ [신청 버튼]  [NEW 뱃지]  │
└──────────────────────────┘
```

- **썸네일**: 16:9 비율 / object-fit: cover
- **제목**: 18px / Bold / Navy
- **설명**: 14px / Gray / 2줄 제한
- **신청 버튼**: Outlined / 주황색 / "자세히 보기"
- **NEW 뱃지**: 빨간색 / 우상단 / "신규"
- **카드 배경**: 흰색
- **모서리**: 12px
- **그림자**: `0 2px 8px rgba(0,0,0,0.1)`

#### 학습 영상 섹션

**섹션 제목**: "🎬 추천 학습 영상" (24px / Bold)

**비디오 카드 구조**:

```
┌──────────────────────────┐
│ [썸네일 + 재생 아이콘]    │
│         ▶               │
│                          │
├──────────────────────────┤
│ [영상 제목]              │
│ 👁️ 1,234 | ⏱️ 12:34     │
└──────────────────────────┘
```

- **썸네일**: 16:9 비율 / 어두운 오버레이
- **재생 아이콘**: 중앙 / 60px / 흰색 / 반투명 배경
- **제목**: 16px / Bold / Navy / 2줄 제한
- **메타 정보**: 14px / Gray
  - 조회수 (눈 아이콘)
  - 재생 시간 (시계 아이콘)
- **Hover**: 재생 아이콘 확대 (scale: 1.1)

#### OX 퀴즈 섹션

**섹션 제목**: "❓ OX 퀴즈로 복습하기" (24px / Bold)

**퀴즈 카드 구조**:

```
┌──────────────────────────┐
│ [문제 번호]              │
│                          │
│ [퀴즈 질문 텍스트]        │
│                          │

- **문제 번호**: 좌상단 / 주황색 / 작은 뱃지 / "Q1"
- **질문**: 16px / Navy / 가운데 정렬
- **O/X 버튼**: 크고 둥근 버튼
  - O: 초록색 (#22C55E)
  - X: 빨간색 (#EF4444)
  - 크기: 80px × 80px
  - 모서리: 완전 라운드
- **점수**: 14px / Bold / 우하단
- **배경**: 연한 회색 (#F9FAFB)

│    │   [비밀번호 입력]          │     │

- **배경**: 흰색
- **모서리**: 16px
- **그림자**: `0 4px 16px rgba(0,0,0,0.1)`

#### 로고 및 제목

- **로고**: "청년있슈" / 48px / Bold / Navy (#1e3a8a)
- **부제**: "로그인하고 청년 정책 혜택을 받아보세요" / 16px / Gray

- **하단 여백**: 32px

**이메일 입력**:
- Label: "이메일"
- Type: email
- Placeholder: "example@email.com"
- Required: ✅
- Validation: 이메일 형식

**비밀번호 입력**:
- Label: "비밀번호"
- Type: password
- Placeholder: "••••••••"
- Required: ✅
- 눈 아이콘: 비밀번호 표시/숨김

**공통 스타일**:
- Component: Quasar Q-Input
- Variant: Outlined
- 높이: 56px
- 여백: 16px

#### 로그인 버튼

- **텍스트**: "로그인"
- **크기**: Full width / 높이 48px
- **배경**: 주황색 (#F97316)
- **텍스트**: 흰색 / 16px / Bold
